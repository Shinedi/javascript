#### linux命令
```
ssh 用户名@192.168.1.1
ls -a //查看文件夹
ll // 查看文件列表
rm -rf 文件名 // 删除
cd 路径
mv 原文件名 新文件名
mv 文件 路径 // 将文件移动到其他目录
cp 原文件名 新文件名 // 拷贝
rm 文件 // 删除文件
touch 文件名 // 新建文件
vi d.js //编辑器
cat 文件名 // 查看文件内容
grep 关键字 文件 // 查案某个关键字
```
